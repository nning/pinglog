# pinglog

Small tool to ping continuously and save log as JSON 

## Usage

    Usage of ./pinglog:
      -c int
            count of packets to be sent
      -d string
            destination (default "1.1.1.1")
      -i int
            output write interval in seconds (default 10)
      -o string
            output path (default "pinglog.json")

## Build

    make
